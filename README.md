# Project Name
PSUSphere

## Short Description
PSUSphere is a Django-based web application that manages student organizations, including colleges, programs, students, organizations, and memberships. It allows administrators to manage data efficiently and generate initial test data for development.

## Features
- Manage colleges and programs
- Manage student information
- Create and manage organizations
- Track student memberships in organizations
- Generate initial data for testing using custom management commands
